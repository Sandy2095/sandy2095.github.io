CSS Version Info:

  • CSS1 - 1996
  • CSS2 - 1998

Syntax:

Selector {property : value}
  • All tags
  • property --> HTML attributes are converted in CSS properties
  • value --> Values are assigned to properties
Ex: table {border : 1px; color:#FFFFFF }

Types of Selectors:

Type Selector --> h1{color : #FFFFFF}

Universal Selector ---> * {color:#23465}

Descendant Selector ---> Apply style to particular element when lies inside particular element , ul em {color:#FFFFFF}

Class Selector --->Based on class attributes .black{color:#ffffff} , h1.black{color:#ffff}

ID Selector --> based on Id attributes of elements , #black{#ffffff}

Child Selector --->similar tot Descendant , direct funcationality of , body > p {color:#ffffff}

Attribute Selector ---> Applies to particular attributes , input[type ="text"] { color : #ffffff } , = -> exactly , ~= -> Contains, |= -> exactly starts or starts with..

Grouping Selector -->many elements with comma seperated , h1,h2,h3 {color : #ffffff}

4 ways to associate styles in HTML:

Embedded CSS --> put css rules into html doc using

Inline CSS --> rule will be appiled to particulat element

External CSS --> element can be used to include external stylesheet file with .css extension,

Imported CSS --> @import used to import external style sheet , @import "URL" , @import url("URL")

Rules Overridding:

  • inline stylesheet takes highest priority override any rule defined in external style sheet file or style tag
  • style defined in style tag will get override with external style sheet file
  • Rules defined in external style sheet is lowest prioirty

CSS Font:

  • Font-Family , Change face of the font
  • Font-Style , Used to make font italic or oblique /* normal , italic, Oblique */
  • Font-variant ,Used to create small caps effect /* normal,small-caps */
  • Font-weight , Used to increase or decrease how bold or light a font appears /* noraml, bold, bolder, lighter, 100,200,...,800,900 */
  • Font-size , Used to increase or decrease the size of the font /* x-small,xx-small,small,medium,large,x-large,xx-large,smaller,larger,size in pixels or in % */
  • Font-shorthand /* style tag font:italic small-caps bold 15px georgia; */

CSS Text:

  • color , set color of text color:red
  • direction , set directon of text , lrt,rtl /* direction:rtl*/
  • letter-spacing ,space between characters , Possible values are normal or number specifying space
  • word-spacing , set the space between words. Possible values are normal or number specifying space
  • text-indent , indent the first line of a paragraph. Possible values are % or number specifying space
  • text-align how to align a text. Possible values are left, right, center, justify.
  • text-decoration how to decorate a text. Possible values are none, underline, overline, line-through, blink.
  • text-shadow how to set the shadow around a text. This may not be supported by all the browsers.
  • white-space how white space inside an element is handled. Possible values are normal, pre, nowrap.
  • text-transform how to set the cases for a text. Possible values are none, capitalize, uppercase, lowercase.
